# AI-based Smart Surveillance Systems

This repository contains notebooks, model pipelines, and curated artifacts for accident detection and violence/weaponized event analysis.

## Main Components

- `pipeline.ipynb`: End-to-end surveillance pipeline notebook.
- `Acidental/main.ipynb`: Accident detection workflow notebook.
- `Acidental/runs/`: Accident detection run outputs (metrics/plots/predictions).
- `Voilence/main2.ipynb`: Violence classification workflow notebook.
- `Voilence/Train/Weaponized` and `Voilence/Test/Weaponized`: Relevant weaponized subset used in this repo.

## Notes on Repository Scope

To keep uploads lightweight and relevant:

- Large model weight files are excluded.
- Full training/validation datasets are excluded where not required.
- Only selected run artifacts and relevant weaponized content are tracked.

See `.gitignore` for exact inclusion and exclusion rules.
